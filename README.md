# gacha_game

トランプカードを使ったガチャゲーム

URL: https://fortegp05.github.io/gacha_game/

## 概要

このプロジェクトは、トランプカードをシャッフルして4枚引き、特定の条件を満たすと「当り」になるガチャゲームです。

## 機能

- トランプカード52枚から4枚をランダムに選択
- 複数の当り条件を設定可能
- Canvasを使用したカード表示
- 当り/ハズレの判定表示
- Twitterシェア機能

## 当り条件

当り条件は`atari.yml`で管理されており、以下の条件が設定されています：

- **フォーカード**: 4枚すべてが同じ数字
- **今日の日付**: カードの数字が今日の日付と一致（例: 12/14なら 1, 2, 1, 4）
- **全て黒**: 4枚すべてが黒（スペードまたはクラブ）
- **全て赤**: 4枚すべてが赤（ハートまたはダイヤ）

将来的には上記以外の当りを追加していきます。

## ファイル構成

```
gacha_game/
├── index.html       # メインのHTMLファイル
├── script.js        # ゲームロジック（カード生成、シャッフル、当り判定）
├── style.css        # スタイルシート
├── atari.yml        # 当り条件の設定ファイル
├── CLAUDE.md        # Claude向けガイドライン
└── README.md        # このファイル
```

## 技術スタック

- HTML5 Canvas（カード描画）
- JavaScript（ゲームロジック）
- CSS（スタイリング）
- js-yaml（YAMLファイルの読み込み）

## 使い方

1. `index.html`をブラウザで開く
2. 「カードをシャッフル」ボタンをクリック
3. 4枚のカードが表示され、当り条件を満たしていれば「当り」が表示される
4. 当りの場合は「もう一度回す」ボタンが表示される
